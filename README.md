## 💡 About the Project
**AI Canvas** is an open-source digital notebook ecosystem built specifically for the **iPad**. It bridges the gap between the natural Apple Pencil handwriting experience and the power of Artificial Intelligence.

The core idea is to provide a minimalist, distraction-free environment where you can organize multiple notebooks, write using Apple's native ink aesthetics, and leverage AI to enhance your learning process.

## ✨ Key Features
* **Native Apple Aesthetics:** Low-latency handwriting optimized for iPad and Apple Pencil.
* **Smart Notebooks:** Create and manage multiple notebooks for different subjects (Engineering, Calculus, etc.).
* **AI Integration:** A built-in sidekick to help you:
    * Perform complex mathematical and engineering calculations.
    * Summarize handwritten or typed notes.
    * Explain difficult concepts or provide additional context.
* **Open Source:** A platform built for students and developers. Fully customizable and transparent.

## 🚀 Getting Started
1. Clone the repository: `git clone https://github.com/lucaspanzera/ai-canvas.git`
2. Open the project in Xcode.
3. Add your AI API Key to the configuration file (see `.env.example`).
4. Build and run on your iPad or Simulator.
